# Cloud Tools

Este diretório agrupa utilitários e scripts para gerenciamento de ambientes em nuvem, com foco em AWS e Azure.

## Subdiretórios

- `aws/` — ferramentas para AWS, incluindo login SSO, configuração de perfis e inventário.
- `azure/` — ferramentas para Azure, especialmente para gerenciar credenciais de AKS e subscriptions.

## Objetivo

Prover scripts e documentação para:

- autenticação e login em contas AWS/Azure
- cadastro e atualização de clusters Kubernetes no `kubeconfig`
- geração de configurações de perfis AWS via Organizations
- busca de recursos na AWS
